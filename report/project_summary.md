# Project summary (portfolio version)

## Goal
Build a reproducible workflow that cleans and integrates Airbnb listings + reviews data and produces decision-oriented visualisations.

## Approach
- Standardised types and handled missingness
- Normalised time fields in reviews for temporal analysis
- Integrated listings with neighbourhood boundaries for location-based insights
- Produced visualisations focused on pricing and neighbourhood comparisons

## Deliverables
- Reusable pipeline under `src/`
- Saved figures under `results/figures/`
- Notebook for transparency under `notebooks/`
